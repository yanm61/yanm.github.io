Hexo + Github

发表于 2015-12-31 | 阅读次数:

安装

  • Node.js
  • Git

如果您的电脑中已经安装上述必备程序，那么恭喜您！接下来只需要使用 npm 即可完成 Hexo 的安装

$ sudo npm install -g hexo-cli

建站

$ hexo init <folder>

$ cd <folder>

$ npm install

关联GitHub

创建仓库

新建名称为您的用户名.github.io的仓库

部署

$ hexo g

$ hexo d

命令

初始化

$ hexo init [folder]

新建文章

$ hexo new [layout] <title>

如果标题含有空格
$ hexo new "post title with whitespace"

生成静态文件

$ hexo generate

启动服务器

$ hexo server

部署网站

$ hexo deploy

清除缓存

$ hexo clean

常见问题

  • 4000端口被占用
    • $ hexo s || hexo s -p 4000
  • 无法连接 git 或找不到 git
    • $ npm install hexo-deployer-git –save
  • 如何设置「阅读全文」
    • 在文章中使用 <!– more –> 手动进行截断

设置主题

如果您想换一个主题的话，可以到Hexo主题挑选自己中意的主题。

这里以nexT 主题主题为例。

  • git clone https://github.com/iissnan/hexo-theme-next themes/next
  • 打开blog目录下的_config.yml，找到theme:修改为theme: next。
